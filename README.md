# Hostel Management System

A modern web application for managing hostel operations, including room allocation, maintenance tracking, and complaint management.

## Features

- **Authentication System**
  - Role-based access control (Admin & Allottee)
  - Secure login with Google
  - Admin login with credentials

- **Dashboard**
  - Overview of hostel statistics
  - Quick access to important features
  - Responsive design for all devices

- **Room Management**
  - View and manage room allocations
  - Track room availability
  - Room status updates

- **Complaint System**
  - Submit and track complaints
  - Admin response system
  - Status updates and notifications

- **Maintenance Tracking**
  - Schedule and track maintenance tasks
  - Priority-based task management
  - Maintenance history

## Tech Stack

- **Frontend**
  - Next.js 14
  - React
  - TypeScript
  - Tailwind CSS
  - Shadcn UI
  - Framer Motion

- **Backend**
  - Next.js API Routes
  - Prisma ORM
  - PostgreSQL
  - NextAuth.js

- **Authentication**
  - NextAuth.js
  - Google OAuth
  - Custom credentials provider
